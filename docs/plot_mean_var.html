plot_mean_varR Documentation

Mean-variance relationship

Description

Plot the mean-variance relationship of transcripts as modeled in sleuth. Each dot represents a transcript. The blue dots represent the transcripts that are used in the shrinkage estimation. The fitted curve represents the smooth fit.

Usage

plot_mean_var(obj, which_model = "full", point_alpha = 0.4,
  point_size = 2, point_colors = c("black", "dodgerblue"),
  smooth_alpha = 1, smooth_size = 0.75, smooth_color = "red")

Arguments

obj

a sleuth object

which_model

which fit to use

point_alpha

the alpha (opacity) of the points (0, 1)

point_size

the size of the points

smooth_alpha

the alpha (opacity) of the line

smooth_size

the size of the line

smooth_color

the color of the smooth line

Details

The x-axis represents the mean expression of transcripts pooled across all samples. The y-axis represents the 'biological' variance after the technical variance has been removed.

Value

a ggplot object

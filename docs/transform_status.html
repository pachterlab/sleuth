transform_statusR Documentation

Check Transform Sync Status of Sleuth Fits

Description

This method prints out the sync status for all fits of sleuth object If the sleuth object's transform function was changed after sleuth_fit was used, the user will need to redo sleuth_fit for any fits already done.

Usage

transform_status(obj)

Arguments

obj

a sleuth object.

Value

a print out of each fit with the transform sync status.
